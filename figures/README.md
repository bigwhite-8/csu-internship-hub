# figures

This folder is reserved for screenshots used in the project `README.md`.

Recommended files to add:
- `home-guide.png`: main guide homepage (hero + knowledge cards)
- `community-board.png`: community page with post list
- `resource-aggregation.png`: internship/employment external resource panel
- `post-modal.png`: post creation modal (image/PDF attachment options)

Guidelines:
- Use PNG format for clarity.
- Suggested width: 1400–1920 px (desktop capture).
- Avoid exposing private information in screenshots.
- Keep filenames stable so README image links remain valid.
